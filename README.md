# Raghee Horner Indicator
Raghee Horner indicator is a trend indicator for MT4/MT5/cTrader coded by EarnForex.com. It was originally developed by Raghee Horner and is used in her famous trading method.

The indicator is based on a three 34-period exponential moving averages and canlesticks' position relative to these EMAs.

It supports multi-timeframe (MTF) operation and all kinds of alerts.
